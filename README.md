# diabetes-risk-prediction-ML
Diabetes Risk Prediction Using Machine Learning

A Machine Learning project for predicting diabetes risk using structured clinical data.
This project is part of my research preparation for applying to PhD positions in AI & Digital Health.

The goal of this project is to analyze a medical dataset and develop machine learning models that can predict diabetes risk based on basic clinical features.

Model building using multiple ML algorithms

Model evaluation and comparison
